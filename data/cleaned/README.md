## Data Availability

The original raw dataset is not included in this repository because the SAS Transport file is large. It can be downloaded from the official CDC BRFSS 2015 Survey Data and Documentation page:

https://www.cdc.gov/brfss/annual_data/annual_2015.html

The full cleaned dataset is also not uploaded because it exceeds GitHub's web upload size limit. It can be regenerated by running the data cleaning notebook.

A small cleaned sample dataset is provided for demonstration.
