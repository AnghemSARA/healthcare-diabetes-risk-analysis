# Healthcare Diabetes Risk Analysis

## Project Overview

This project focuses on healthcare data cleaning, exploratory data analysis, and diabetes risk analysis using the CDC BRFSS 2015 survey dataset. The main objective is to prepare a clean and structured dataset, explore important health patterns, and prepare the data for future machine learning prediction.

## Dataset Source

The raw dataset comes from the CDC Behavioral Risk Factor Surveillance System (BRFSS) 2015.

Official source: https://www.cdc.gov/brfss/annual_data/annual_2015.html

The original dataset contains 441,456 records and 330 variables. In this project, selected healthcare, lifestyle, and demographic variables were extracted and cleaned.

## Data Availability

The original raw dataset is not included in this repository because the SAS Transport file is large. It can be downloaded directly from the official CDC BRFSS 2015 Survey Data and Documentation page:

https://www.cdc.gov/brfss/annual_data/annual_2015.html

The full cleaned dataset and the EDA-ready dataset may also be too large for direct GitHub web upload. They can be regenerated by running the notebooks provided in this repository.

## Data Cleaning Summary

The cleaning process was based on the official CDC BRFSS 2015 raw dataset and codebook. The objective was to prepare a clean and structured dataset for diabetes risk analysis while avoiding unnecessary data loss.

The cleaning process included:

* Selecting relevant healthcare, lifestyle, and demographic variables from the raw BRFSS dataset.
* Creating a binary diabetes target variable from `DIABETE3`.
* Removing unclear diabetes target responses to keep only confirmed diabetes and non-diabetes cases.
* Checking unclear and missing values before applying the final cleaning strategy.
* Handling missing and unclear values according to each variable type and missing-value percentage.
* Keeping unknown income responses as a separate category to avoid unnecessary data loss.
* Keeping unknown cholesterol-check and physical-activity responses as separate categories because of their relatively high missing or unclear rates.
* Converting BMI from `_BMI5` into readable BMI values and imputing missing BMI values using the median.
* Transforming alcohol consumption into estimated drinking days per month and imputing missing values using the median.
* Transforming fruit and vegetable consumption into estimated daily frequency and imputing missing values using the median.
* Removing old raw coded columns after creating clean replacement features.

## Final Cleaned Dataset

Final cleaned dataset shape:

```text
379,170 rows × 21 columns
```

The final cleaned dataset contains no missing values.

## Exploratory Data Analysis Summary

The exploratory data analysis step was performed to understand the cleaned dataset before machine learning.

The EDA process included:

* Checking dataset shape, columns, data types, missing values, and duplicated rows.
* Analyzing the target variable `Diabetes_binary`.
* Studying numerical features such as BMI, physical health days, mental health days, alcohol consumption, fruit consumption, and vegetable consumption.
* Studying categorical features such as general health, cholesterol check, heart disease, stroke, sex, education, income, smoking, physical activity, and age group.
* Visualizing distributions using bar charts, histograms, and boxplots.
* Calculating diabetes rates by important groups such as age group, BMI category, general health, physical activity, income, heart attack history, and stroke history.
* Checking correlations with the diabetes target.
* Recoding selected yes/no variables when needed to make correlation interpretation clearer.
* Capping extreme food-frequency values at 10 times per day to reduce unrealistic extreme values.
* Saving an EDA-ready dataset for the next machine learning step.

## Main EDA Findings

The EDA showed that the dataset is imbalanced:

```text
No diabetes: 86.89%
Diabetes: 13.11%
Imbalance ratio: 6.63 : 1
```

The main patterns observed during EDA were:

* Diabetes rate increases with age.
* Diabetes rate increases with BMI.
* Poor general health is associated with a higher diabetes rate.
* No physical activity is associated with a higher diabetes rate.
* Lower income groups show higher diabetes rates.
* Heart attack history and stroke history are associated with higher diabetes rates.
* Important associated features include general health, difficulty walking, BMI, age group, physical unhealthy days, heart disease or heart attack, physical inactivity, and stroke.
